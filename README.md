## VaultEdge Finance

VaultEdge Finance is a modern fintech website that's designed to help young professionals manage their money effectively. The platform focuses on budgeting, savings, and financial awareness through a clean and user-friendly interface.

# Project Overview
  
This project is a multi-page website built as part of an HTML and CSS assignment. It demonstrates front-end development skills, layout structuring, and responsive design principles.
VaultEdge Finance aims to simulate a real-world fintech product by presenting features such as financial tracking, smart savings, and user-focused design.

# Collaboration

This project was completed through collaboration between two team members, working together on design, structure, and implementation.

# Technologies used
- HTML5

# Pages Included
- Home Page
- About Page

# Features
- Clean and modern UI design

- HTML files for each page
